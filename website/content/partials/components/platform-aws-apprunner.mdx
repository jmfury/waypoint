## aws-apprunner (platform)

Creates an App Runner service, with the specified configuration
from `waypoint.hcl`.

### Interface

### Required Parameters

These parameters are used in the [`use` stanza](/docs/waypoint-hcl/use) for this plugin.

#### name

- Type: **string**

### Optional Parameters

These parameters are used in the [`use` stanza](/docs/waypoint-hcl/use) for this plugin.

#### cpu

- Type: **int**
- **Optional**

#### memory

- Type: **int**
- **Optional**

#### port

- Type: **int**
- **Optional**

#### region

- Type: **string**
- **Optional**

#### role_name

- Type: **string**
- **Optional**

#### static_environment

- Type: **map of string to string**
- **Optional**
